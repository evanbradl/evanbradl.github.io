Police Blotter

Data Analysis

Project to track police activity in Iowa City with a focus on alcohol related arrests. This project takes data about arrests from the Iowa City arrests blotter and adds the info to a excel sheet originally recieved from the IC police department with info about alcohol realated arrests from 1/1/2014-5/6/2019. Info pulled from police blotter includes date, time, location, charge, name of arrestee, name of officer making arrest and DOB of arrestee. The location is then taken and using google maps api translated to latitude and longitude and also stored in excel doc. See pictures for examples of graphs program can produce.
